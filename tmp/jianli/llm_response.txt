### 项目经验  

#### 发票大师 —— 企业级智能费控与发票管理平台  
**技术栈：** SpringBoot、SpringCloud、MyBatisPlus、MySQL、MongoDB、Redis、Kafka、Eureka、Feign、Seata、Apollo、Activiti、Aviator、XxlJob、CAT、Keycloak、AirFlow、Metabase  

**项目简介：**  
一款面向企业用户的智能化财务管理系统，覆盖发票采集、验真、报销、预算控制、企业支付、记账凭证生成等全流程。目前已服务超 8 万家企业，累计用户超 40 万人，日均处理单据量达百万级。  

**核心职责与成果：**  
1. **系统架构设计：** 参与微服务拆分与技术选型，搭建基于 SpringCloud 的微服务架构，使用 Eureka 实现服务注册发现，Feign 完成服务间通信，Apollo 统一配置管理。  
2. **单据模块设计：** 基于 DDD 领域模型设计单据核心领域，采用 CQRS 架构分离查询与命令逻辑；命令操作通过 Kafka 异步处理，提升系统响应速度与解耦能力。  
3. **动态表单引擎：** 设计可拖拽配置的动态表单系统，使用访问者模式实现表单解析器，将表单结构与业务逻辑解耦；配置托管至 Apollo，支持热更新。  
4. **权限控制体系：** 基于注解实现细粒度字段级权限控制，支持按用户、单据类型、审批状态动态控制表单控件的读写权限。  
5. **工作流引擎定制：** 使用 Activiti 实现审批流引擎，扩展 UserTask 节点支持签收、抄送、支付等场景；通过 ExecutionListener 实现业务回调；优化长流程定义解析性能，避免参数截断问题。  
6. **分布式事务保障：** 采用 Seata AT 模式保障单据状态与工作流状态的一致性，确保关键业务数据的强一致性。  
7. **风控规则引擎：** 基于 EasyRules + Aviator 搭建动态风控系统，拓展 Aviator 支持 Spring Bean 调用，实现规则表达式中访问上下文数据；规则配置热更新，支持用户自定义报销规则，拦截超标、违规单据，审批效率提升 40%。  
8. **企业支付集成：** 对接支付宝、连连支付企业付 API，实现事前费控能力；通过 SQL 优化与 Redis 缓存，将费控校验耗时控制在 100ms 内。  
9. **记账凭证自动化：** 根据企业财务规则，将报销单据自动转换为标准记账凭证，推送至外部财务系统，实现财务流程自动化。  
10. **预算管控体系：** 从 0 到 1 设计预算全生命周期管理框架，涵盖预算编制、控制、调整、分析与考核；采用 Event Sourcing 模式记录预算变更事件，确保金额可追溯、数据准确，满足跨国集团复杂管控需求。  
11. **数据可视化：** 使用 AirFlow 定时同步业务数据至 Metabase，构建多维度财务分析看板，支持管理层决策。  
12. **统一认证平台：** 集成 Keycloak 实现 OAuth2 认证与权限管理，支撑运营后台与多租户身份体系。  
13. **平台打通：** 实现与钉钉、飞书的深度集成，自动同步审批状态与通知，保障跨平台数据一致性。

#### 快打单 —— 高并发电商打单系统  
**技术栈：** SpringBoot、MyBatisPlus、ShardingJDBC、MySQL、PolarDB、RocketMQ、Redis、Eureka、Prometheus、Grafana、SkyWalking、Sentinel、XxlJob、K8S、Docker、Jenkins  

**项目简介：**  
服务于电商商家的智能打单工具，支持多平台订单同步、自动拆合单、电子面单打印、进销存管理等功能。日均同步订单超 1000 万单，日均打单发货超 100 万单，系统稳定运行于 K8S 集群。  

**核心职责与成果：**  
1. **多平台接入：** 集成快手、抖音、京东、微信、魔筷等平台 OpenAPI，通过适配器模式解耦平台差异，支持快速拓展新渠道；使用 XxlJob 定时刷新 Token，保障授权长期有效。  
2. **订单同步重构：** 重构订单同步模块，解决高并发下漏单问题；引入 ShardingJDBC 实现订单表分库分表，优化查询与写入性能；结合 JVM 调优，整体同步速率提升 20%，数据准确率达 100%。  
3. **快递面单集成：** 对接顺丰、中通、极兔等快递平台电子面单接口，实现自动化打单发货，日均打单量超 100 万单。  
4. **智能拆合单：** 实现基于收货人、地址、商品等维度的自动拆合单逻辑，帮助商家降低物流成本。  
5. **备货单生成：** 结合进销存系统，自动生成备货清单，提升仓储准备效率。  
6. **短信通知集成：** 打通运营商短信通道，支持商家自定义模板，自动在打单时触发用户通知。  
7. **系统监控：** 集成 SkyWalking 实现服务调用链追踪；使用 Prometheus + Grafana 构建监控告警平台，设计核心指标（QPS、延迟、错误率）实现异常自动预警。  
8. **流量防护：** 引入 Sentinel 实现接口限流与熔断，防止恶意刷单或攻击导致服务雪崩。  
9. **数据库迁移：** 将订单数据从 MySQL 迁移至 PolarDB，解决阿里云 RDS 存储瓶颈，降低 30% 机器成本。  
10. **CI/CD 与容器化：** 使用 Docker 打包服务，通过 Jenkins 实现自动化构建与部署至 K8S 集群，结合监控指标实现弹性伸缩，进一步优化资源利用率。

#### 授权爬虫框架 —— 分布式用户数据采集平台  
**技术栈：** SpringBoot、Consul、MySQL、OTS、OSS、Redis、Kafka、Apollo、Selenium WebDriver、Prometheus  

**项目简介：**  
基于用户授权的分布式爬虫框架，用于采集淘宝、支付宝、网银、运营商等平台的用户行为与账单数据，日均处理任务超 10 万次，数据用于风控与信用评估。  

**核心职责与成果：**  
1. **反爬突破：** 通过分析 JS 加密逻辑、模拟登录流程，结合 Selenium WebDriver 虚拟浏览器技术，成功绕过淘宝、支付宝等平台的签名验证与滑块验证机制。  
2. **配置热更新：** 设计动态配置加载机制，爬取规则与解析逻辑托管至 Apollo，支持在线修改并实时生效，无需重启服务即可应对目标网站前端变更。  
3. **弹性调度：** 基于 Prometheus 监控任务队列长度，结合 Consul 服务发现实现自动扩缩容，高峰期动态增加爬虫节点，低峰期自动回收，节约 40% 服务器成本。  
4. **数据清洗与存储：** 使用 Kafka 解耦采集与解析流程，数据经清洗后存储至 OTS（表格存储）与 OSS（对象存储），保障高可用与可追溯性。
